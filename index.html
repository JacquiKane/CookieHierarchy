Now For Something Different! Cookie Inheritance Workshop

The plain cookie class has all the basic ingredients. A chocolate chip cookie class just includes another ingredient - chocolate chips. Another cookie, nut chocolate chip cookie, differs from the chocolate chip cookie, with added nuts. It makes sense for ChocolateChipCookie class to inherit from PlainCookie class. NutChocolateChipCookie would inherit from ChocolateChipCookie class. A 3 layer hierarchy!

Select cookie, click Construct and call the constructor. Instance method buttons are disabled until the cookie class is instantiated.
